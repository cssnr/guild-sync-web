About

This service synchronizes World of Warcraft Classic guild members to Discord Users and Roles...

There is currently 2 methods for syncing members.

  1. Name Based: Matches In-Game names to Discord usernames. Requires you to keep Discord usernames up to date with In-Game names.
  2. Note Based: Matches In-Game note to Discord usernames. Requires all members to set an In-Game note with their Discord username.

Current Features

  • Can select a "Guild Role" to apply to all matched members.
  • Can enable syncing "Class Roles" to all matched members.

Coming Soon

  • Can select a Discord channel to receive alerts for all changes made by this service.
  • Can enable auto creation of class roles in Discord if they do not exist.

For more information Join our Discord.
