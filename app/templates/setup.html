Setup Docs

1. Enable Guild Sync

Log in to the website, select a server from the home page, click the Enable button, and follow the dialogs to add the bot to your server.

2. Select Sync Settings

Log in to the website, select a server from the home page, enable the desired options.

3. Update Role in Discord

Log in to Discord, go to the server, open the server settings page, go to Roles and drag the "Guild Sync" role above all other roles it needs to manage.

Discord will not let anyone modify any roles that are above it even if they are an administrator.

4. Install WoW Addon

Download and install the World of Warcraft Classic Addon.

Extract the archive to your Addons folder.
Example: C:\Program Files (x86)\World of Warcraft\_classic_\Interface\AddOns

  • Download: Guild Sync Addon
  • Source/Docs: Guild Sync Addon

Make sure the Addon is enabled and log in to World of Warcraft. Wait 30 seconds for a full sync.
You can verify the addon is working and do a manual sync with: /gs sync

5. Install Client App

Download and run the Guild Sync Client App.

  • Download: Guild Sync Client
  • Source/Docs: Guild Sync Client

You will need your Client Access Key found in your Profile from the dropdown in the top-rgiht corner of the website.
